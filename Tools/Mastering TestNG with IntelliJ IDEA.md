- TestNG is a testing framework inspired by JUnit and NUnit.
- Key benefits:
  - Supports annotations
  - Enables test configuration and grouping
  - Generates detailed reports
- Setup:
  - Install TestNG plugin in IntelliJ IDEA
  - Add TestNG library to your project



---



### 2Ô∏è‚É£ Create & Run Tests in TestNG + Reports
- Create test classes using `@Test` annotation.
- Execution flow:
  - IntelliJ IDEA runs tests via TestNG XML or directly from the class.
- Reports:
  - HTML and XML reports generated by default
  - Useful for analyzing test outcomes and failures



---



### 3Ô∏è‚É£ Annotations in TestNG
- Common annotations:
  - `@BeforeSuite`, `@AfterSuite`
  - `@BeforeTest`, `@AfterTest`
  - `@BeforeClass`, `@AfterClass`
  - `@BeforeMethod`, `@AfterMethod`
  - `@Test`
- Purpose:
  - Control setup and teardown logic
  - Define test execution lifecycle



---



### 4Ô∏è‚É£ Prioritizing & Disabling Tests
- Use `priority` attribute in `@Test` to control execution order.
- Disable tests using `enabled = false`:
  ```java
  @Test(enabled = false)
  ```



---



### 5Ô∏è‚É£ Dependency Tests
- Use `dependsOnMethods` to create test dependencies:
  ```java
  @Test(dependsOnMethods = {"loginTest"})
  ```
- Ensures dependent tests run only if prerequisites pass.



---



### 6Ô∏è‚É£ Grouping Tests
- Use `groups` attribute to categorize tests:
  ```java
  @Test(groups = {"smoke", "regression"})
  ```
- Run specific groups via TestNG XML or command line.



---



### 7Ô∏è‚É£ Assertions
- Validate test outcomes using:
  - `Assert.assertEquals()`
  - `Assert.assertTrue()`
  - `Assert.assertFalse()`
- Helps verify expected vs actual results.



---



### 8Ô∏è‚É£ Parameters in TestNG
- Pass parameters via TestNG XML:
  ```xml
  <parameter name="browser" value="chrome"/>
  ```
- Use `@Parameters` annotation in test methods.



---



### 9Ô∏è‚É£ DataProvider in TestNG
- Enables data-driven testing:
  ```java
  @DataProvider(name = "loginData")
  public Object[][] getData() { ... }
  ```
- Use with `@Test(dataProvider = "loginData")`



---

- Improves execution speed and resource utilization.
